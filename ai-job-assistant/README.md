# AI Job Assistant

## V1 Features
- Resume and job description match scoring
- Matched and missing skills detection
- Resume improvement suggestions
- Tailored summary generation
- Resume bullet improvement

## Run V1 backend
```bash
uvicorn app.v1_main:app --reload
```

## Run frontend
```bash
streamlit run ui/streamlit_app.py
```

## Run tests
```bash
pytest
```
